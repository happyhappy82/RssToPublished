# 해외 소스 종합 마케팅 툴 - 개발 계획서

## 프로젝트 개요
해외 AI 분야 저명 인사들의 콘텐츠를 자동 수집하고, AI로 가공하여 한국 스레드/링크드인에 업로드하는 종합 마케팅 대시보드

---

### 1.1 개발 환경 구성
- [ ] Next.js 14 (App Router) 프로젝트 생성
- [ ] TypeScript 설정
- [ ] Tailwind CSS + shadcn/ui 설치
- [ ] ESLint, Prettier 설정
- [ ] 프로젝트 폴더 구조 생성

### 1.2 데이터베이스 설정
- [ ] Supabase 프로젝트 생성
- [ ] 환경 변수 설정 (.env.local)
- [ ] 데이터베이스 스키마 설계 및 마이그레이션
  - [ ] `sources` 테이블 (해외 계정 정보)

  - [ ] `prompts` 테이블 (저장된 프롬프트)

  - [ ] `upload_queue` 테이블 (업로드 대기열)
- [ ] Supabase 클라이언트 설정
